Narrate & bind

Set everything here, then start once and leave it — it narrates every chapter, then (if you choose) combines them into one audiobook and builds the YouTube video, all in a single run.

Flush RAM & page file every chapters, then continue — keeps memory flat so long books finish cleanly.

If narration already finished (even in a previous session) but binding failed or was skipped, this combines the chapter files already on disk — no re-narrating. It uses the combine / MP4 / cover options selected above.

Runs entirely on your machine. Cloud engines send text to your chosen provider; local engines send nothing.
